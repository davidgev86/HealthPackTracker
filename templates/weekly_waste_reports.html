Archive Information

Automatic Archival

The waste log is automatically archived every 7 days. When this happens:

  • Current waste log entries are moved to the archive folder
  • A weekly report is generated with totals and breakdowns
  • The waste log is cleared to start fresh

This process ensures your data is preserved while keeping the active log manageable.

Manual Archive

Use the "Force Archive" button to manually archive current data for testing or administrative purposes.
